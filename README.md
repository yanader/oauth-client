# OAuth2 Client

This is a simple project used to explore how OAuth2 can be used to provide secure login functionality. It works in tandem with [this Spring backend](https://github.com/yanader/oauth-server).

## How to run this repo:

### Clone the repo

`git clone https://github.com/yanader/oauth-client.git`

### Install dependenies

`npm i`

### Locally run the client

`npm start`

Then visit http://localhost:3000 in your browser.
